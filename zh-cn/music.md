## 快捷键
!> `F5` `开始`播放选中的歌曲

!> `F6` `继续`播放歌曲

!> `F7` `暂停`播放歌曲

!> `F8` `停止`播放歌曲

!> `F2` 切换到`下一首`

## 乐器的选择
!> 如果是 `长按类` 的乐器，请将`延音设置为至少0.6以上`
## 间隔延迟
间隔延迟自带为 `0s`，随机 `0.01 - 0.06` 区间，自定义设置最低 `0` 最高 `2`
## 延音设置
延音系统自带为 `0.01s`，随机 `0.02 - 0.6` 区间，自定义设置最低 `0` 最高 `2`
## 倍速设置
倍速设置最低为 `0.5` 最高 `5`

## 上传自己的谱子

!> 目前支持的谱格式为`.txt`
<br />
内容为
``` json
[
	{
		"name": "demo",
		"bpm": 200,
		"bitsPerPage": 16,
		"pitchLevel": 0,
		"isComposed": false,
		"songNotes": [
			{
				"time": 862,
				"key": "1Key5"
			}
		]
	}
]
```
